# Pipeline

One-off Python scripts live here, per PLAN.md §6: clip GHSL rasters to Attica
and vectorize epoch bands, parse ELSTAT construction-period tables and join
them to boundaries from geodata.gov.gr, extract and simplify OSM footprints
via Geofabrik, and convert Urban Atlas GeoPackage classes.

Scripts are run once and their outputs committed to `data/`. There is no
server component and no CI beyond the GitHub Pages deploy workflow.

Empty as of Stage 2 (scaffold). Populated in Stage 3.
